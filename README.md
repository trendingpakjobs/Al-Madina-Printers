# Al Madina Printing Press Website

Official website for **Al Madina Printing Press**, a trusted printing service provider in Mansehra, Pakistan since 1998. The website showcases their high-quality printing solutions for schools, businesses, and government institutions.

## 🌐 Features

- Fully responsive design (Desktop & Mobile)  
- Premium animations on hero, cards, and buttons  
- Hero section with professional printing image  
- 3 main blocks: Institutional Printing, School Stationery, Business Printing  
- Services section with clickable cards & icons  
- Stats counter section  
- Featured products section  
- Portfolio section with professional images/videos  
- Floating WhatsApp button & footer contact links  
- SEO optimized for all pages  
- Disclaimer page with legal content  
- Dark blue, gold, and white color theme for premium look  

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (for animations & card interactions)  
- Responsive design (Flexbox/Grid)  
- SEO best practices
